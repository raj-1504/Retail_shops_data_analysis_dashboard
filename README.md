# Retail_shops_data_analysis_dashboard
GOAL: Understanding how weather and customer demographics impacts performance and provide insights to improve planning, staffing and promotions.

Approach:
  -step 1: Gather and clean daily sales, weather and survey data (Build a database).
  -step 2: Join datasets on date for unified analysis (Develop SQL).
  -step 3: Explore descriptive statistics and visualizations (Analyze data).
  -step 4: Create Powerbi dashboards for stakeholders.

Analysis Objective:
 - Link sales data with weather patterns and survey demographics.
 - Identify correlations; temperature vs. sales, rainfall effects, weekend's lift.
 - Compare shop performance and seasonal patterns.
 - Build foundation dashboards and predictive modelling.

Expected Outcomes:
  - Clear picture of how weather drive customer demand.
  - Insight of family vs. single, male vs. female shopping patterns.
  - Dashboard that allows ongoing monitoring of trends.
